# Python Implementation of Ensemble Modulation [Bishop & Hodyss (2008)]

This directory contains an implementation of Bishop and Hodyss' Ensemble Modulation (EM). The original reference is
```
Bishop, C., & Hodyss, D. (2009). Ensemble covariances adaptively localized with ECO-RAP. Part 2: A strategy for the atmosphere. Tellus A: Dynamic Meteorology and Oceanography, 61(1), 97-111.
```

This implementation is written by M-Y Chan
